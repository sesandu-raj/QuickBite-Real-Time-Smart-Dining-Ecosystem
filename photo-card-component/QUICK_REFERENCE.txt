## Quick Customization Checklist

```
For Any Card:
[ ] Update background-image URL
[ ] Change title/name text
[ ] Update description/quote
[ ] Change icon (if applicable)
[ ] Update price (if applicable)
[ ] Add onclick link (if clickable)
[ ] Check image size (800x600+ recommended)
[ ] Test on mobile
```

## Copy-Paste Template

### Single Food Card
```html
<div class="food-card glass animate-on-scroll">
  <div class="card-img" style="background-image: url('https://YOUR_IMAGE_URL')"></div>
  <div class="card-content">
    <div style="display: flex; justify-content: space-between">
      <h3>YOUR ITEM NAME</h3>
      <span class="price">$X.XX</span>
    </div>
    <p>YOUR DESCRIPTION HERE</p>
    <button class="btn-order" style="width: 100%">Add to Order</button>
  </div>
</div>
```

### Single Team Card
```html
<div class="team-card glass animate-on-scroll">
  <div class="portrait-wrapper">
    <div class="portrait" style="background-image: url('https://YOUR_PHOTO_URL')"></div>
    <div class="role-icon">
      <span class="material-symbols-outlined">YOUR_ICON</span>
    </div>
  </div>
  <h3>NAME HERE</h3>
  <span class="role-title">JOB TITLE</span>
  <p class="quote">"THEIR QUOTE HERE"</p>
</div>
```

### Single Category Card
```html
<div class="category-card glass" onclick="location.href = 'your-page.html'">
  <div class="cat-img" style="background-image: url('https://YOUR_IMAGE_URL')"></div>

      <h3>CATEGORY NAME</h3>
    </div>
    <p>YOUR DESCRIPTION</p>
  </div>
</div>
```
