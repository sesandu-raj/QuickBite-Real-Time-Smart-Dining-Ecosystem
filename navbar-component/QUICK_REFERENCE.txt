## ðŸ”‘ Key Points to Remember

1. **Always use relative paths** - `../` goes up one folder
2. **Update active links** - Current page link gets `class="active"`
3. **Keep same CSS/JS** - All pages use same `style.css` and `script.js`
4. **Don't edit navbar CSS** - It's in the main `style.css`
5. **Don't edit navbar JS** - It's in the main `script.js`
